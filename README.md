# ZeeTetris
A simple javascript game based on Tetris Attack (SNES)

## Controls
|KEY       | ACTION                 |
|----------|------------------------|
|UP        | move aim up            |
|DOWN      | mode aim dow           |
|LEFT      | move aim left          |
|RIGHT     | move aim right         |
|ENTER     | start game             |
|SPACE     | reverse block position |
|BACKSPACE | restart game           |
